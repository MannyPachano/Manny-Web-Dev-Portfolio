Project Overview

When Panda's Playhouse reached a user base of 5000 active members, they decided it was time to upgrade from a simple Facebook page to a full-fledged membership based website.


Research showed that to really take the next step forward, their site would require a facial verification system for KYC verification, a dollar-to-credit conversion system for their virtual currency, a mobile-friendly design as 95% of their users used mobile devices, and fast site speeds to ensure user satisfaction.


Following an initial discussion to better grasp their needs and concerns, we decided that building the site on a drag-and-drop platform, commonly known as a WYSIWYG CMS, would be the most suitable approach. This choice would empower their team to independently make updates and edits to the website in the future without needing to hire outside developers.


The owners asked if I'd be able to build their site using Wix, as they'd had previous experience with this platform. After some research, I found that Wix recently launched Wix Studio, a product designed to give developers more flexibility on their site builds by allowing access to CSS and 'Velo' - Wix's version of Javascript.


I dove in head first, researching online casino competitors, and drew up some initial wireframes to get a good handle on how the user's journey should flow across the site. Following the UX, I created a beautiful UI to boost confidence in the site's credibility and made sure to match their logo's colors. This was all showcased using Figma's prototyping tools.


The development was a bit of a challenge as I needed to quickly learn the Velo syntax and have the site completed by our agreed upon due date. Thanks to unlimited coffee and more sleepless nights than I'd care to admit, I came up to speed with the syntax and managed to complete the site on schedule.


The site features a third-party API that guides users through a KYC verification process with facial recognition, a virtual currency system implemented using Wix Studio's native tools, complete responsiveness across screen sizes, as well as fast site speeds accomplished by optimizing the site's images and code. Panda's Playhouse owners were ecstatic when they previewed their new site, and even more-so now following the site launch as they are able to make their own website edits using Wix's user friendly WYSIWYG system.


PS: As Panda's Playhouse has gone through many in-house updates and edits since it's inception, with some of these edits having had a poor impact on UI and site speed, I chose to remove live site viewing as the current site does not represent the aforementioned finished product.
