Class Tracker Web App

A web application designed for friends to gamify their martial arts journey. Features include EXP progress bars and rad characters as you level up.

Project Overview

My friends and I needed a way to motivate ourselves to show up to our martial arts classes. I decided this was the perfect opportunity to practice my skills and create a web app that would gamify our experience.


I used HTML, CSS, and Javascript for the frontend, and Google's firebase database to keep track of our progress across devices. The app is 100% responsive across screen sizes and features animated progress bars that increase every time you attend a class (honor system folks). Classes increase your EXP points according to their (subjective) difficulty.


As you level up, watch as your character evolves into something more badass - representative of your newfound skills.

Tools Used

  • HTML
  • CSS
  • JavaScript
  • SQL
  • Google Firebase
  • Github
  • Netlify
